# 🚀 WorkSmart AI – AI-Powered Workplace Productivity Assistant

## Project Overview

WorkSmart AI is a modern AI-powered workplace productivity assistant designed to help professionals automate repetitive tasks, improve efficiency, and make better decisions. The application leverages Artificial Intelligence to streamline daily workplace activities such as email writing, meeting summarization, task planning, research assistance, and productivity analysis.

The platform acts as a centralized workplace assistant that helps users save time, stay organized, reduce burnout, and increase overall productivity through intelligent automation and personalized recommendations.

### Problem Statement

Professionals often spend significant time on repetitive administrative tasks such as drafting emails, managing schedules, summarizing meetings, and researching information. These activities reduce time available for high-value work and can contribute to workload stress.

WorkSmart AI addresses this challenge by providing AI-powered tools that automate common workplace tasks while offering actionable productivity insights and workload management recommendations.

## Setup Instructions

### Prerequisites

Ensure the following are installed:

* Node.js (v18 or later)
* npm or yarn
* Git

### Clone the Repository

```bash
git clone https://github.com/yourusername/worksmart-ai.git

cd worksmart-ai
```

### Install Dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env` file:

```env
OPENAI_API_KEY=your_openai_key
GEMINI_API_KEY=your_gemini_key
SUPABASE_URL=your_supabase_url
SUPABASE_ANON_KEY=your_supabase_key
```

### Start Development Server

```bash
npm run dev
```

Application will run at:

```text
http://localhost:3000
```

### Build for Production

```bash
npm run build
```
